# Glossary

## Adapter

Infrastructure component that translates between an external API or device protocol and OpenInputHub domain concepts.

## Capability

A declaration of what a device or plugin can do, such as `Has6DOF`, `HasButtons`, or `CanOutputHID`.

## Device Session

A runtime connection to a physical, virtual, replayed, or network-backed device.

## Event Bus

Runtime mechanism that distributes normalized events between producers and consumers.

## InputEvent

The normalized event model used inside the OpenInputHub core.

## Plugin

Replaceable extension unit for input, output, transforms, recording, replay, profiles, or diagnostics.

## Replay

Injection of previously recorded normalized events into the runtime as a virtual input source.
